# 2026-02-28

## OpenClaw ノート同期設計（新構成）

### 設計方針
1. workspace 以下に `document/read` と `document/write` を作成する。
2. `document/read` は iCloud 側とシンボリックリンクで同期する。
3. OpenClaw は `document/write/note/OpenClaw` 以下にのみ書き込みを行う。
4. `document/write/note/OpenClaw` に書き込みがあった場合、何らかの手段で iCloud 側の `note/OpenClaw` にコピーする。

※ サンドボックス制限を考慮した分離設計。
