# OpenClawノート symlink同期設定（2026-02-28）

## 🎯 目的
OpenClawが作成するノートを、iCloud（Obsidian）側へリアルタイム反映させる。

---

## 📂 ディレクトリ構成

### 元（workspace側）
```
/Users/tbtsub/.openclaw/workspace/millerDoc/note/OpenClaw
```

### 同期先（iCloud / Obsidian）
```
/Users/tbtsub/Library/Mobile Documents/com~apple~CloudDocs/Obsidian/note/OpenClaw
```

---

## 🛠 実施手順

### ① 既存ディレクトリ削除（symlink化のため）
```
rm /Users/tbtsub/.openclaw/workspace/millerDoc/note/OpenClaw
```

※ 実体ではなくリンクのみ削除

---

### ② symlink作成
```
ln -s \
"/Users/tbtsub/Library/Mobile Documents/com~apple~CloudDocs/Obsidian/note/OpenClaw" \
"/Users/tbtsub/.openclaw/workspace/millerDoc/note/OpenClaw"
```

---

### ③ 設定確認
```
ls -l /Users/tbtsub/.openclaw/workspace/millerDoc/note
```

表示例：
```
OpenClaw -> /Users/.../Obsidian/note/OpenClaw
```

`->` が表示されていれば成功。

---

## ⚠ 注意点

- `rsync --delete` は symlinkを削除する可能性がある
- 双方向同期はループの原因になる
- symlink方式では cron同期は原則不要

---

## ✅ 現在の設計

- OpenClaw → iCloud：symlink（リアルタイム）
- 逆方向cron：削除済み

---

## 🧠 メモ

構成をシンプルにするほど事故は減る。
リアルタイム同期が必要な場合はsymlinkが最小構成。
